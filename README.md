## Ex-6 UI & UX 

### Aim:
To perform UX design activities including solution ideation, user story creation, scenario development, flow mapping, and information architecture design for improving user experience in digital applications.

### Algorithm:
#### 1. Solution Ideation
Identify a UX problem (e.g., hospital appointment booking).
Conduct a Crazy 8s or Brainwriting session.
Generate multiple solution ideas within a limited time.
Sketch low-fidelity concepts for each idea.
Select the most feasible solution.

#### 2. Creating User Stories
Define user personas and their goals.
Identify user needs and challenges.
Write user stories using the format:
As a [user], I want to [goal], so that [benefit].
Validate stories against user requirements.
Prioritize the user stories.

#### 3. Creating Scenarios
Select a user story.
Define the user's environment and context.
Describe the user's actions and interactions.
Include emotional states during task completion.
Review the scenario for completeness.
 
#### 4. Flow Diagrams / Flow Mapping
Select a user task (e.g., ordering food online).
Identify all actions and decision points.
Arrange steps sequentially.
Use standard flowchart symbols to create the flow.
Verify the flow for logical consistency.

#### 5. Information Architecture (IA)
Choose a website or application.
Conduct card sorting to group content.
Categorize information into logical sections.
Create a sitemap representing the structure.
Evaluate the navigation hierarchy.
